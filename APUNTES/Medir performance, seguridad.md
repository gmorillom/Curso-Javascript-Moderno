## Consejos de seguridad

1. No almacenar contraseñas en el localStorage
2. Utilizar Dom Scripting la mayoría del tiempo, solo utiliza innerHTML cuando tengas certeza que los datos ingresados no son maliciosos
3. Utiliza JWT o Json Web Token
4. Cuando trabajes con dependencias utiliza sitios web como snyk.io que te dará información sobre vulnerabilidades que pueda presentar la dependencia en alguna de sus versiones
5. Utiliza hashing para encriptar datos sensibles como contraseñas
